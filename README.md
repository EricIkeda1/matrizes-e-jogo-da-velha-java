# Atividades com Matrizes em Java

Este repositório contém a implementação de exercícios práticos utilizando matrizes em Java, com foco no desenvolvimento do raciocínio lógico e na manipulação de estruturas bidimensionais.

## Objetivo

Praticar conceitos fundamentais de programação, como:

- Estruturas de repetição (`for`)
- Manipulação de matrizes
- Entrada e saída de dados
- Operações matemáticas com matrizes
- Lógica de programação

---

## Atividades Desenvolvidas

### 1. Preenchimento e exibição de matriz 3x3
Programa que recebe valores inteiros do usuário e exibe a matriz preenchida.

### 2. Contagem de números negativos em matriz 5x5
Lê uma matriz e informa quantos elementos são negativos.

### 3. Soma de duas matrizes 4x4
Realiza a soma entre duas matrizes e exibe o resultado.

### 4. Diagonal principal de matriz 4x4
Mostra os elementos da diagonal principal.

### 5. Matriz transposta 3x3
Exibe a matriz transposta, trocando linhas por colunas.

### 6. Multiplicação de matrizes (2x3 × 3x2)
Calcula o produto entre duas matrizes compatíveis.

### 7. Verificação de matriz simétrica 3x3
Verifica se a matriz é igual à sua transposta.

### 8. Média por linha e coluna (3x4)
Exibe a matriz e calcula:
- Média de cada linha  
- Média de cada coluna  

---

## Desafio: Jogo da Velha

Implementação do jogo da velha utilizando uma matriz 3x3.

### Funcionalidades

- Dois jogadores (X e O)
- Alternância de jogadas
- Validação de posições ocupadas
- Verificação de vitória (linha, coluna ou diagonal)
- Detecção de empate
